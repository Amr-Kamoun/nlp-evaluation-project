# Evaluate a News Article with Natural Language Processing

## Project Description
This project evaluates news articles using Natural Language Processing (NLP) to extract polarity, subjectivity, and text snippets.

## How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Use `npm run start` for development or `npm run build` for production.

## Dependencies
- Node.js
- Express
- Webpack
- Jest
